
# special models
These routers smartly filters for models that support features needed for your request such as image understanding, tool calling, structured outputs and more.

model `openrouter/free`
- a router that selects free models at random from the models available on OpenRouter.

model `openrouter/auto`
- a router that selects paid models available on OpenRouter

# Limit
- embedding models are not currently supported.
  - No embedding endpoints in OpenRouter's API
